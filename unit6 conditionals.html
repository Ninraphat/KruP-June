Section B — Reading Passage

An article using all four conditional types. Spot them all!

🎨
BLUE = Type 0  ·  GREEN = Type 1  ·  PURPLE = Type 2  ·  RED = Type 3

Decisions, Decisions

~190 words · Advisory Article · B1

Making good decisions is a skill everyone needs. If you heat water to 100°C, it boils — cause and effect are simple in science. But in life, decisions are rarely so straightforward.

If you study hard this month, you will pass the entrance exam. It's that simple. Many students, however, delay studying. If they don't start now, they won't be ready in time. Remember: if you prepare well, you won't regret it.

Some students dream of a different life. "If I had more time, I would practise English every day," they say. But the truth is, we all have 24 hours. If I were you, I would make a study schedule right now.

Looking back is also important. "If I had started earlier, I would have had more time to prepare," one student admitted. "If my teacher had explained it more clearly, I might have understood it better."

The lesson? Don't wait. If you act today, you will thank yourself later.

Section C — Grammar Spotlight: All 4 Conditional Types

Click each type to expand. Every type includes formula, meaning, examples, and Thai notes.

🔧
Structure of every conditional sentence:
If-clause (condition)  +  Main clause (result)
The if-clause and main clause can swap positions. When the if-clause comes SECOND, no comma is needed.
"She will pass if she studies." (no comma needed)
0

Type 0 — Zero Conditional

General truths, scientific facts, things that are always true

Always True ›
If + Present Simple,   Present Simple
🔥
If you heat ice, it melts.
ถ้าคุณให้ความร้อนแก่น้ำแข็ง มันเลียเต็ม (จริงเสมอ)
🌎
If it rains, the ground gets wet.
ถ้าฟนตก พื้นดินเปียก (ความจริงทุกครั้ง)
🔧
If you press this button, the door opens.
ถ้าคุณสดปุ่มนี้ ประตูเปิด (พฮติกรรมทุกครั้ง)
💡
Note: You can also use "when" instead of "if" in Type 0: "When you heat ice, it melts." The meaning is the same for facts.
1

Type 1 — First Conditional

Real and possible future situations

Real Future ›
If + Present Simple,   will + V1
📚
If you study hard, you will pass the exam.
ถ้าคุณตั้งใจเรียน คุณจะสอบผ่าน (เป็นไปได้จริง)
⛈
If it rains tomorrow, we won't go on the trip.
ถ้าแล่วมีฟนพรุ่งนี้ เราไม่ไป
📱
If you call me, I will answer.
ถ้าคุณโทรนุนลนมา หนูสะออก
⚡
Variations in the main clause:
Instead of will, you can use: can, may, might, should, or an imperative.
"If you finish early, you can leave." / "If you're hungry, eat something."
❌
Common Error: "If you will study, you will pass." ✗
NEVER use "will" in the if-clause of Type 1. Use Present Simple instead.
2

Type 2 — Second Conditional

Imaginary / hypothetical present or future situations

Unreal Present ›
If + Past Simple,   would/could/might + V1
🌌
If I had more money, I would travel the world. (But I don't have much money.)
ถ้านโย෽มีเงินมากกว่านี้ นโย෽คงเดินทางรอบโลก (แต่ความเป็นจริง: โย෽ไม่มีเงินมาก)
🤔
If I were you, I would apologise. (But I am not you.)
ถ้านโย෽เป็นคุณ นโย෽คง෾อโทษ (แต่เป็นไปไม่ได้)
🐦
If she could fly, she would visit every country.
ถ้าเธอบินได้ เธอคงเยียมทุกประเทศ
⚡
Key rule: Use "were" for ALL subjects in Type 2
"If I were…" / "If she were…" / "If he were…"
In informal English, "was" is sometimes heard, but "were" is always correct in exams.
3

Type 3 — Third Conditional

Imaginary / impossible past situations — regret or criticism

Unreal Past ›
If + Past Perfect (had + V3),   would/could/might + have + V3
📚
If I had studied, I would have passed. (But I didn't study, so I didn't pass.)
ถ้านโย෽เรียน นโย෽ผ่าน (แต่จริง: ไม่เรียน ไม่ผ่าน)
🚜
If she had left earlier, she wouldn't have missed the train.
ถ้าเธอออกเดินทางเร็วกว่านี้ เธอไม่พลาดรถไฟ
💼
If he had applied sooner, he might have got the job.
ถ้าเ෾ายื่นสมัครเร็วกกว่านี้ เ෾าอาจได้งาน
💡
Type 3 = always talking about the PAST (impossible to change).
The real situation is the OPPOSITE of what is stated.
"If I had studied" = I did NOT study.
"I would have passed" = I did NOT pass.
❌
Most Common Error:
"If I would have studied, I would have passed." ✗
NEVER use "would have" in the if-clause. Use "had + V3".
✓ "If I had studied, I would have passed."

Quick Reference — All 4 Types

TypeIf-ClauseMain ClauseMeaningReality
Type 0 Present Simple Present Simple General truth / scientific fact Always true
Type 1 Present Simple will + V1 Real future possibility Possible & likely
Type 2 Past Simple (were) would/could/might + V1 Imaginary present/future Untrue / unlikely
Type 3 Past Perfect (had + V3) would/could/might + have + V3 Imaginary past (regret) Impossible (past)

Special Connectors: Unless, As long as, Provided that

ConnectorMeaningExample
unless= if…not (except if)"Unless you study, you will fail." = "If you don't study, you will fail."
as long as= if and only if / on the condition that"You can go out as long as you finish your homework."
provided (that)= on the condition that (formal)"I will help you, provided that you help me in return."
in case= as a precaution (take action now in case something happens later)"Take your umbrella in case it rains." (NOT "if it rains")
even if= despite the condition (the result is the same)"Even if you apologise, she won't forgive you."

Section D — Mixed Conditionals

Combining different time frames — a favourite in advanced exams.

🔧
What is a Mixed Conditional?
A mixed conditional combines the if-clause from one type with the main clause from a different type. This happens when we want to mix a PAST condition with a PRESENT result, or vice versa.
M1

Mixed Type 1: Past Condition → Present Result

If something had happened in the past, the present would be different now

›
If + Past Perfect (had + V3)  ← Type 3 if-clause,   would + V1  ← Type 2 result
📚
If I had studied medicine, I would be a doctor now.
ถ้านโย෽เรียนแพทย์ในอดีต ตอนนี้นโย෽เป็นหมอดูแล้ว (แต่ผลคือพฯตีกรรมปัจจุบัน)
🌎
If she had taken that job, she would be living in New York now.
ถ้าเธอรับงานนั้น ตอนนี้เธอคงอยู่ที่นโย์คแล้ว
M2

Mixed Type 2: Present Condition → Past Result

If something were true now, the past would have been different

›
If + Past Simple (were)  ← Type 2 if-clause,   would + have + V3  ← Type 3 result
🧠
If I were braver, I would have spoken to her yesterday.
ถ้านโย෽แกร่าวกว่านี้ เมื่อวานนี้นโย෽คงฒีกคุยกับเธอแล้ว
📊
If he were more careful, he wouldn't have made that mistake.
ถ้าเ෾าระมัดระวังยิ่งกว่านี้ เ෾าคงไม่ทำข้อผิดพลาดนั้น
⚡
Memory Trick for Mixed Conditionals:
M1 = Past Condition → Present Result: had + V3  ,  would + V1
M2 = Present Condition → Past Result:  were / V2  ,  would have + V3
The tenses in each clause refer to DIFFERENT time frames.
✏

Section E — Exercises

Five activities covering all conditional types — from identification to exam format.

1

Identify the Conditional Type

Choose which conditional type is used in each sentence.

📌 Look at the verb forms in both clauses to identify the conditional type.
1 "If you freeze water, it turns into ice."
A Type 0 — general truth
B Type 1 — real future
C Type 2 — unreal present
D Type 3 — unreal past
✓ Type 0: If + Present Simple ("freeze"), Present Simple ("turns"). This is a scientific fact — always true.
✗ Both verbs are Present Simple — that's Type 0 (general truth).
2 "If I won the lottery, I would donate half of it to charity."
A Type 0
B Type 1
C Type 2
D Type 3
✓ Type 2: If + Past Simple ("won"), would + V1 ("would donate"). Imaginary present/future — winning the lottery is unlikely.
✗ "Won" (Past Simple) in if-clause + "would donate" in main clause = Type 2 (unreal present).
3 "If she had taken the medicine, she would have recovered faster."
A Type 0
B Type 1
C Type 2
D Type 3
✓ Type 3: If + Past Perfect ("had taken"), would + have + V3 ("would have recovered"). Impossible past — she did NOT take the medicine.
✗ "Had taken" (Past Perfect) + "would have recovered" = Type 3 (impossible past regret).
4 "If you don't hurry, you'll miss the bus."
A Type 0
B Type 1
C Type 2
D Type 3
✓ Type 1: If + Present Simple ("don't hurry"), will + V1 ("will miss"). Real future possibility — this could actually happen.
✗ "Don't hurry" (Present Simple) + "will miss" = Type 1 (real future possibility).
5 "If she had been more careful, she would be safe now."
A Type 2
B Type 3
C Mixed (Past Condition → Present Result)
D Type 1
✓ Mixed Conditional (M1): If + Past Perfect ("had been") = past condition, would + V1 ("would be safe now") = present result. Two different time frames!
✗ Past Perfect if-clause + "would + V1" (not "would have + V3") = Mixed Conditional. The "now" is the key clue.
6 "Unless you apologise, she will never forgive you."
A Type 0
B Type 1 (with "unless")
C Type 2
D Type 3
✓ "Unless" = "if…not". "Unless you apologise" = "If you don't apologise". The verb forms match Type 1 (Present Simple + will).
✗ "Unless" = "if…not". The verb forms (Present Simple + will) show Type 1.

Fill in the Correct Verb Form

Use the type given in brackets to choose the correct form.

📌 Identify the conditional type, then use the correct verb form. Click Reveal to check.
1. If it rains (rain) tomorrow, we will cancel the picnic. [Type 1]
✓ rains — Type 1: If-clause uses Present Simple, even though we're talking about the future.

✓ were — Type 2: Use "were" for ALL subjects (I, he, she, it, we, they).
3. If you had arrived (arrive) on time, you wouldn't have missed the presentation. [Type 3]
✓ had arrived — Type 3: If-clause uses Past Perfect (had + V3).
4. Water boils if you heat (heat) it to 100°C. [Type 0]
✓ heat — Type 0: Both clauses use Present Simple. This is a scientific fact.
5. If she had practised (practise) more, she could have won the competition. [Type 3]
✓ had practised — Type 3: If-clause = Past Perfect. Result = could have + V3.
6. If I had learnt (learn) Chinese as a child, I would speak it fluently now. [Mixed — M1]
✓ had learnt — Mixed Conditional M1: Past condition (had + V3) + present result (would + V1 "would speak… now").

✓ apply — "Unless" = "if…not". Type 1: unless + Present Simple + will + V1.
8. If she were (be) more confident, she would have asked for a promotion last year. [Mixed — M2]

3

Error Correction

Each conditional sentence has ONE error. Find and fix it.

📌 Click "Show Correction" to see the fix and rule.
1. If you will study harder, you will pass the exam. ✗ will study
✓ If you study harder, you will pass the exam.
Type 1: NEVER use "will" in the if-clause. Use Present Simple instead.
2. If I was rich, I would travel the world. ✗ was
✓ If I were rich, I would travel the world.
Type 2: Use "were" for ALL subjects in the if-clause, not "was".
3. If I would have studied, I would have passed. ✗ would have studied
✓ If I had studied, I would have passed.
Type 3: NEVER use "would have" in the if-clause. Use Past Perfect (had + V3).
4. Unless you don't hurry, you'll miss the train. ✗ don't hurry
✓ Unless you hurry, you'll miss the train.
"Unless" already means "if…not". Adding "don't" creates a double negative. "Unless you hurry" = "If you don't hurry".
5. If she hadn't been ill, she would attend the meeting. ✗ would attend
✓ If she hadn't been ill, she would have attended the meeting.
Type 3: if-clause uses Past Perfect (hadn't been), so result clause must use "would have + V3".
6. If he studies yesterday, he would pass today. ✗ studies
✓ If he had studied yesterday, he would pass today. (Mixed M1)
The if-clause refers to the past ("yesterday"), so it needs Past Perfect (had + V3). "Would pass" (present result) shows it's a Mixed Conditional M1.

Sentence Transformation

Rewrite each sentence using the conditional type given.

📌 Keep the same meaning but change the sentence to the conditional type shown in brackets.
1. "She doesn't study, so she fails." → Rewrite as Type 1:
If she studies, she will pass / won't fail.
✓ If she studies, she will pass / won't fail. (Type 1: Present Simple + will + V1)
2. "He isn't a doctor, so he can't treat patients." → Rewrite as Type 2:
If he were a doctor, he could treat patients.
✓ If he were a doctor, he could treat patients. (Type 2: Past Simple "were" + could + V1)
3. "She didn't wear a coat, so she caught a cold." → Rewrite as Type 3:
If she had worn a coat, she wouldn't have caught a cold.
✓ If she had worn a coat, she wouldn't have caught a cold. (Type 3: Past Perfect + would not have + V3)
4. "He didn't save money in the past, so he's poor now." → Rewrite as Mixed (M1):
If he had saved money, he would be rich now.
✓ If he had saved money, he would be rich now. (Mixed M1: Past Perfect + would + V1)
5. "You don't believe in yourself, so you didn't try last time." → Rewrite as Mixed (M2):
If you believed in yourself, you would have tried last time.
✓ If you believed in yourself, you would have tried last time. (Mixed M2: Past Simple + would have + V3)

Section F — Unit Review

Summary of all conditional types + self-evaluation checklist.

🚨
Top 8 Exam Traps (ข้อสอบชอบออก):
  1. "If you will study…" ✗ → "If you study…" ✓ — NEVER "will" in Type 1 if-clause
  2. "If I was rich…" ✗ → "If I were rich…" ✓ — Use "were" for ALL subjects in Type 2
  3. "If I would have studied…" ✗ → "If I had studied…" ✓ — NEVER "would have" in if-clause
  4. "Unless you don't study…" ✗ → "Unless you study…" ✓ — "unless" already means "if not"
  5. Confusing Type 2 and Type 3: "would + V1" (Type 2 present) vs "would + have + V3" (Type 3 past)
  6. Forgetting "have" in Type 3 result: "would passed" ✗ → "would have passed" ✓
  7. Using wrong tense in "in case": "in case it will rain" ✗ → "in case it rains" ✓
  8. Mixed Conditionals: confusing M1 (had + V3 → would + V1) and M2 (were/V2 → would have + V3)
