Nouns 🍎

คำนาม: ฉบับครอบคลุมพื้นฐาน (ป.1-6)

1. คำนามคืออะไร?

คำนาม (Noun) คือคำที่ใช้เรียกชื่อ คน สัตว์ สิ่งของ และสถานที่

2. นามทั่วไป VS นามชี้เฉพาะ

• Common Noun: เรียกกว้างๆ (dog, boy, city)
• Proper Noun: ชื่อเฉพาะ (Snoopy, Tom, Bangkok) ➡️ ขึ้นต้นด้วยตัวพิมพ์ใหญ่เสมอ

3. นับได้ VS นับไม่ได้

• นับได้: มี 1, 2, 3 ชิ้นได้ (apple, car)
• นับไม่ได้: ของเหลว, ผง, เงิน (water, milk, money) ➡️ ห้ามเติม a/an/s

4. เอกพจน์ และ พหูพจน์

• เอกพจน์ (1 ชิ้น): a cat
• พหูพจน์ (2 ชิ้นขึ้นไป): cats
• กฎเปลี่ยนรูป: เติม s, es (boxes), ies (babies), เปลี่ยนร่าง (man -> men, child -> children)
